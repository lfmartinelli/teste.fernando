Properties
connectionString
static string
String of connection to access ODBC database
Set this before to instantiate obejct.
The value in this property is global and is inherit for every instances of smartObj
data
Dictionary[string, dynamic]
Data of first register in dataList. Using [nameData] you identify data collection The fields be the properties of obejct
Ex: myobject.data["my_nameData"].name, myobject.data["my_nameData"].age, etc
dataBase
string
Which database sever plataform your project wil be use? SQL Server, PostgreSQL, MySQL, Firebase etc.
(not working in this moment)
dataCollections
Dictionary[string, dynamic]
Data collections informations. The dataName is the name of dataCollection. Identify dataColletion by the dataName.
Ex: myobject["clients"].fields
Property "tableName" - Name of the table in database used to load query registers in the data collection.
Property "fields" - Relation of fields separate by coma in query results.Ex: name,age,address
Property "fieldKey" - Principal primary key of your data table.
Property "rowCount" - Number of rows in query.
Property "colCount" - Number of collumns in query.
This field is the reference to update register or insert new register, so can't result null or repeated value becouse this.
Ex: myobejct.dataTables["client"].rows[2]["name"], myobejct.dataTables["products"].rows[2]["price"]
dataCollumns
Dictionary[string, Dictionary[string, dynamic]]
Data collumns informations maxLengh and dataType of collumns.
Identify collumns collections by dataName.
Property "maxLenght" - Max lenght of charcteres determined to collumn.
Property "dataType" - Type of data in collumn.
Ex: myobject.dataCollumns["users"]["age"].maxLenght, myobject.dataCollumns["clients"]["name"].dataType
dataPage
int
Page of data of SELECT return. The number of rows in return query be limited by maxRows value
Each data page load registers collection with maxRows limits of rows.
Ex: If maxRows=100 => dataPage=1 is row 0 to 99; dataPage 2 is row 100 to 199
dataTables
Dictionary[string, DataTable]
Data tables with query result when query result have more than 1 row.
You can identify table by dataName.
Ex: myobejct.dataTables["client"].rows[2]["name"], myobejct.dataTables["products"].rows[2]["price"]
document
String
Document(s) such as contract, sale propose or others to recive register value in [fields].
The document must be in HTML format and fields in [field] format.
Ex: The client [name], and your wife [wife_name].
err
int
Error number if occurres any exception in action
htmlForm HTML
string
code to render form (DIV) with html controls to show register values
htmlGrid
string
HTML code to render html grid control with values of registers in getListWithHTMLGrid method
htmlInput
string
HTML code to render html input control with value of register in getValueWithHTMLInput method
lastAction
DataTime
Date of last action executed by the object
maxRows
int
Maximum number of rows for each data page in query return.
This limit is important to administrate trafic of data in network and cliente memory
The object load only one dataPage rows at one time, so if configurate maxRow to 100, only 100 rows will be stored at one time in dataTable. To exemple, if you want load register in row 110, must change dataPage to number 2
message

Error message if occurres any exception in action

script
string
JavaScript script to execute that in return results to JS by API.
Start and end each function with "[scrpt]" label.
Ex: [script] function fun01(){ instructions...} [script] [script] function fun02(){ instructions...} [script]
When result of API action come back to JavaScript, that script will be executed.
If you using SimpleAPI, that will execute automatically
